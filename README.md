# Monorepo Labs Skills

Agent skills for [Monorepo Labs](https://github.com/monorepo-labs) projects.

## Install

```bash
# Install all skills
npx skills add monorepo-labs/skills

# Install a specific skill
npx skills add monorepo-labs/skills --skill supacortex
```

## Skills

| Skill | Description |
|-------|-------------|
| [supacortex](./supacortex/) | Bookmarks, conversation memory, and identity for [Supacortex](https://supacortex.ai) |
